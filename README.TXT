The application "Vote no restaurante" was developed using the followinf technologies

- Spring Boot
- Spring Data
- Spring JDBC
- JPA
- Apache Velocity
- H2 Database
- Apache Tomcat


Requirements to run the application 

- JDK 1.8 
- Maven 3.3.3

 Assumptions
 
 - No web security was taken into account
 - No performance test
 - No availability test


Deployment

- Go to the sources root folder
- Type the Maven command:

mvn spring-boot:run
